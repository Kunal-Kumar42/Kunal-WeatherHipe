Weather Project

This Weather Web App provides real-time weather information for cities around the world. Users can search for any city to view the current temperature, weather conditions, humidity, wind speed, and atmospheric pressure. The interface is designed to be clean and user-friendly, with an interactive search feature and a responsive layout for desktop and mobile devices. This project demonstrates practical skills in HTML, CSS, and JavaScript, including DOM manipulation, event handling, and API integration for dynamic content. It also emphasizes creating an intuitive user experience while implementing essential front-end web development techniques.
